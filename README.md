# Weather App

Android	architecture	and	Components:	
1.	MVP Architecture.	
2.	RxJava	
3.	Butterknife	
4.	SQLiteDbOpener	for	persisting	data	
5.	Java
6.	Glide	for	Image	
7.	Retrofit	for	Network	Access
8.	AndroidX
